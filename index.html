Maps: An Web app looking just like iOS5 Maps for iOS6 or other devices

To try, click here!

If you are iOS device user, we recommend "Add to Home Screen" to enjoy full screen display.

Screenshots

Same looking, same UI as the original iOS5 Maps.

Japanese localization is also available. It is automatically detected.

Features

Almost all features of iOS5 Maps are implemented.

  • Basic map functions (with display options such like satellite, traffic, and etc.)
  • Current location trace
  • Dropped pin
  • Place search
  • Route search (The availability of transit search is depending on your country/region)
  • Search history
  • Route navigation
  • Bookmark
  • Street View near dropped pin or bookmark

What we are missing now are,

  • Trace accuracy by pulsating
  • Heading trace
  • Display and choise from plural route candidates
  • Detail list of route

Libraries and platforms we use

  • Google Maps JavaScript API V3
  • jQuery
  • Twitter Bootstrap
  • GLYPHICONS FREE
  • Here We Are Again - plebeosaur.us
